
# Use Case

Название: Публикация товара на маркетплейсе  

Актор: Продавец  

Предусловия:   
1. Продавец зарегистрирован на платформе
2. Продавец авторизован в личном кабинете  

Основной сценарий:  
1. Продавец открывает маркетплейс
2. Система загружает главную страницу
3. Продавец открывает личный кабинет
4. Продавец переходит в раздел "Мои товары"
5. Продавец нажимает кнопку "Добавить товар"
6. Система отображает форму создания товара
7. Продавец заполняет форму создания товара
8. Продавец нажимет кнопку "Опубликовать"
9. Система проверяет корректность введенных данных
10. Система сохраняет товар в базе данных
11. Товар становится виден на маркеплейсе  

Альтернативные сценарий:  
1. Продавец нажимает кнопку "Опубликовать"
2. Система обнаруживает некорректность введенных данных
3. Система выводит сообщение об ошибке
4. Продавец заного заполняет поле и повторяет публикацию  

Постусловия: Продавец и клиенты маркетплейса видят новый товар на витрине маркетплейса
