# User Story

Как продавец маркетплейса  
я хочу публиковать свой товар  
чтобы покупатели могли его видеть и покупать.

---

# Acceptance Criteria

1. Продавец может создать карточку товара
2. Продавец может указать наименование товара
3. Продавец может указать описание товара
4. Продавец может указать категорию товара
5. Продавец может указать цену товара
6. Продавец может указать количество товара
7. Продавец может добавить фото товара
8. После публикации товар отображается на маркетплейсе
